## 🛠️ How It Works

StockAI gathers headlines from the internet in real time. It covers markets in the United States and China. The app uses artificial intelligence to scan these headlines. It looks for positive or negative sentiment. This shows you how the market feels about specific companies. You see a clear summary instead of clutter.

The app supports several sources for intelligence. These include models from OpenAI, Anthropic, and DeepSeek. You can also run models on your own machine using Ollama. This keeps your data private.

## ⚙️ Setting Up Your AI

The first time you open the app, you need to link an AI provider. This is how the app understands the news.

1. Open the "Settings" menu in the top corner.
2. Choose your preferred AI source.
3. If you use OpenAI or Anthropic, enter your API key. You can find this key on the provider's website.
4. Save your changes.

The app is now ready to analyze news. You only need to do this step once.

## ⌨️ Using the Interface

The main screen shows a list of stocks. You can search for a company name or a stock symbol. When you select a stock, the app shows a news feed. Each headline features a sentiment score. 

* Green markers indicate positive news.
* Red markers indicate negative news.
* Grey markers indicate neutral outlooks.

You can click any headline to read the full report. The AI summarizes the key points for you. This allows you to scan hundreds of stories in seconds.

## 🛡️ Privacy and Safety

Your data stays on your machine. The app does not save your financial records. It only pulls public news data from the web. When you enter an API key, the app uses it only to talk to the AI provider. It does not share your key with other programs.

## ❓ Common Questions

Can I run this without an internet connection?
No. The app needs the internet to pull news headlines in real time.

What happens if I lose my API key?
You can generate a new key on your AI provider's website. You then update the key in the StockAI settings menu.

Does the app cost money to use?
The app is free to download. Please note that AI providers may charge small fees based on how much data you process. Check your provider's terms for details.

How do I remove the app from my computer?
Go to your Windows Settings, select Apps, find StockAI in the list, and select Uninstall. This removes all files from your system.

What if the app stops responding?
Close the app and restart it. If problems persist, visit the release page to see if an update is available. You can download the newer version to replace the old files.
